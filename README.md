# 🌈 `theme-csx`

A **type-safe**, **scalable**, and **persistent** theming library for React Native & Expo. Built for performance, DX, and cross-platform compatibility.

## ✨ Features

- ⚡️ Light / Dark / System theme support
- 🍏 Dynamic iOS color adaptation
- 💾 MMKV-based persistent theme storage
- 🧠 Memoized themed styles with `useThemedStyles`
- 🔐 Type-safe access to theme tokens (with autocomplete)
- 🚀 Perfect for monorepos and multi-app setups
- 🧩 Extendable (spacing, typography, shadows, etc.)

> ✅ **iOS**: theme changes apply instantly  
> 🔁 **Android**: theme changes apply on app restart

## 📦 Installation

```bash
# npm
npm install theme-csx

# yarn
yarn add theme-csx

# pnpm
pnpm add theme-csx
```
